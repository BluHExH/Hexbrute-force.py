Proxy Support কাজ: Tor (Anonymity) সাপোর্ট ব্যবহার করে ট্র্যাকিং এড়ানো যাবে। এবং: সুরক্ষিত এবং অ্যানোনিমাসভাবে ব্রুট ফোর্স করা যাবে।
